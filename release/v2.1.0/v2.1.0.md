# Oneiro v2.1.0 发行说明

## ✨ 新特性 (New Features)
- **常用网站 (Quick Sites)**: 新增常用网站侧栏！你可以自由添加、编辑和删除常用网站，支持自动获取网站图标（favicon）。
- **智能动态空间自适应**: 书签区和常用网站区现在能智能感知屏幕高度，自动限制最大显示行数并在超出时折叠，无论添加多少个书签或网站，页面都不会出现破坏美感的滚动条。
- **全新空间留白美学**: 重构了页面的垂直对齐逻辑（4:2的弹性留白系统），当页面元素丰富时，诗词能依然稳居视觉C位，整体布局更加沉稳优雅。

## 💄 界面与体验优化 (UX & UI Improvements)
- **设置面板升级**: 重新设计了设置面板，将设置项分组为「外观」和「显示」，并为各项功能加入了直观的开关按钮（Toggle）。
- **洗牌算法 (Shuffle Bag) 优化诗句随机性**: 彻底告别连续随机到同一句诗的体验，现在的算法保证你在看完所有（逾六百首）诗作之前，绝不重复！
- **细节间距微调**: 精确打磨了各个标题、组件之间的CSS Margin，强化了「标题-内容」的视觉绑定感。
