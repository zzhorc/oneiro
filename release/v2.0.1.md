# 🔧 Oneiro v2.0.1

> 修复书签图标加载，新增统一设置面板与书签图标风格切换

## 🐛 修复

- 🔖 **修复 favicon 无法加载** — `chrome://favicon2/` 在 Manifest V3 下被禁止访问，改用扩展自身的 `_favicon/` 端点
- 🖼️ **修复 favicon 模糊** — 请求尺寸从 64px 提升到 128px，高分屏下图标更清晰

## ✨ 新增功能

- ⚙️ **统一设置面板** — 左下角齿轮按钮，点击弹出毛玻璃菜单，整合所有设置项
  - 🌗 主题切换（浅色 / 深色 / 跟随系统）
  - 🔤 字体切换
  - 📊 书签显示行数（1~4 行）
  - 🖼️ 书签图标风格切换
- 🎨 **书签图标三种风格** — 彩色网站图标 / 黑白网站图标 / 首字图标，设置自动持久化

## 🔧 优化

- 🧹 **设置按钮整合** — 原先左下角三个独立按钮合并为统一的设置面板，界面更简洁
- 💾 **设置持久化** — 书签图标风格偏好保存到 localStorage
